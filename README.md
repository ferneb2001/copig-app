# Sistema COPIG - Consejo Profesional de Ingenieros y Geólogos

## Descripción
Sistema de gestión integral para el Consejo Profesional de Ingenieros y Geólogos (COPIG) de Mendoza.

## Tecnologías
- **Backend:** Node.js + Express
- **Base de datos:** PostgreSQL  
- **Frontend:** HTML + JavaScript vanilla
- **Puerto:** 3030

## Funcionalidades Principales
- ✅ Gestión de profesionales y matrículas
- ✅ Gestión de empresas y representantes técnicos
- ✅ Sistema de certificados CHP (Certificados de Habilitación Profesional)
- ✅ Sistema financiero bidireccional con carga de comprobantes PDF
- ✅ Portal unificado con reconocimiento automático de usuarios
- ✅ Panel administrativo completo

## Instalación
```bash
npm install
node server.js
```

## URLs Principales
- Panel admin: http://localhost:3030/admin
- Portal profesional: http://localhost:3030/
- Gestión empresas: http://localhost:3030/empresas

## Estado del Sistema
Sistema completamente funcional con base de datos PostgreSQL integrada.
