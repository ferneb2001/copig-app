# 📋 ANÁLISIS COMPLETO - CERTIFICADO DE HABILITACIÓN PROFESIONAL (CHP) COPIG MENDOZA

## 🎯 ¿QUÉ ES EL CHP?

El **Certificado de Habilitación Profesional** es un documento emitido por el COPIG (Consejo Profesional de Ingenieros y Geólogos de Mendoza) que:

- **Acredita** que el profesional está matriculado y habilitado para ejercer
- **Verifica** que tiene la matrícula al día (pagada)
- **Confirma** que no tiene sanciones éticas pendientes
- **Habilita** para presentar trabajos ante organismos públicos y privados
- **Certifica** que cumple con los requisitos de la Caja Previsional Técnica

## 📁 DOCUMENTOS REQUERIDOS PARA CHP

### 1. **DOCUMENTACIÓN DE MATRÍCULA**
- ✅ Comprobante de pago de matrícula anual COPIG
- ✅ Comprobante de pago de matrícula mensual/trimestral (si corresponde)
- ✅ Certificado de libre deuda con COPIG

### 2. **DOCUMENTACIÓN CAJA PREVISIONAL**
- ✅ Comprobante de pago CAMO (Contribución Anual Mínima Obligatoria) a la Caja Previsional Técnica
- ✅ Certificado de cumplimiento previsional
- ✅ Declaración jurada de obras/trabajos realizados

### 3. **DOCUMENTACIÓN TÉCNICA (según el tipo de trabajo)**

#### Para obras civiles/construcción:
- 📐 **Planos** (arquitectura, estructura, instalaciones)
- 📝 **Memoria descriptiva técnica**
- 📊 **Cómputo y presupuesto**
- 🏗️ **Especificaciones técnicas**
- 📍 **Plano de ubicación**

#### Para instalaciones eléctricas:
- ⚡ **Memoria técnica de instalación eléctrica**
- 📋 **Planillas de carga**
- 🔌 **Esquemas unifilares**
- 📐 **Planos de instalación**

#### Para estudios ambientales:
- 🌍 **Informe de impacto ambiental**
- 📊 **Estudios de suelo**
- 💧 **Análisis de agua**
- 🏭 **Plan de gestión ambiental**

#### Para trabajos de geología:
- 🗺️ **Mapas geológicos**
- 🔬 **Informes de prospección**
- ⛏️ **Estudios de factibilidad minera**
- 📊 **Análisis de muestras**

### 4. **DOCUMENTACIÓN ADMINISTRATIVA**
- 📄 **Nota de solicitud dirigida al COPIG**
- 🆔 **Copia DNI del profesional**
- 📧 **Datos de contacto actualizados**
- 🏢 **Datos del comitente (si aplica)**

## 💰 FORMAS DE PAGO

### Matrícula COPIG:
- Pago anual completo (con descuento)
- Pago en cuotas (mensual/trimestral)
- Transferencia bancaria
- Pago presencial
- Pago online (MercadoPago, etc.)

### Caja Previsional:
- CAMO anual
- Aportes por obra declarada
- Planes de financiación

## 🔄 FLUJO DEL PROCESO CHP

```
1. SOLICITUD INICIAL
   ↓
2. CARGA DE DOCUMENTACIÓN
   - Matrícula al día
   - Caja Previsional al día
   - Documentación técnica según tipo
   ↓
3. REVISIÓN COPIG
   - Verificación de pagos
   - Revisión técnica
   - Control ético/disciplinario
   ↓
4. APROBACIÓN/RECHAZO
   - Si falta documentación → Solicitar
   - Si todo OK → Aprobar
   ↓
5. EMISIÓN DEL CHP
   - Generación del certificado
   - Firma digital/sello
   ↓
6. ENTREGA
   - Descarga PDF
   - Envío por email
   - Retiro presencial
```

## 📱 SISTEMA PROPUESTO

### MÓDULO PROFESIONAL
1. **Solicitud múltiple**: Poder crear varias solicitudes CHP para diferentes obras
2. **Carga diferenciada de documentos**: Según tipo de obra/trabajo
3. **Estado de cuenta integrado**: Ver deudas de matrícula y caja
4. **Seguimiento en tiempo real**: Ver estado de cada solicitud
5. **Historial completo**: Todas las solicitudes anteriores

### MÓDULO ADMINISTRACIÓN
1. **Panel de control**: Ver todas las solicitudes pendientes
2. **Verificación automática**: Cruce con base de datos de pagos
3. **Revisión técnica**: Herramientas para revisar documentación
4. **Comunicación**: Sistema de observaciones/rechazos con motivos
5. **Emisión**: Generación automática del CHP con numeración
6. **Estadísticas**: Reportes de CHPs emitidos por tipo, profesional, etc.

## ⚠️ CONSIDERACIONES IMPORTANTES

1. **Sin matrícula al día = No CHP**
2. **Sin CAMO pagado = No CHP**
3. **Documentación incompleta = Solicitud en espera**
4. **CHP tiene validez limitada** (generalmente 30-60 días)
5. **Cada obra/trabajo requiere su propio CHP**
